# PKL 전체 사용자 플로우

> Personal Knowledge Library — 내 PDF 책을 AI와 함께 읽고 지식을 쌓는 독서 앱  
> 배포: https://pkl-webapp.vercel.app

---

## 4. 핵심 사용자 여정

### 여정 A — 책 읽기 + 실시간 AI 활용

```
[서재 📚] 책 선택
    │
    ▼
[뷰어 📖] PDF 표시
    │
    ├─▶ 페이지 이동: ←/→ 키, 마우스 휠, 모바일 스와이프
    │
    ├─▶ 텍스트 드래그 선택
    │       ├─ 💡 즉석 AI 질문 팝업
    │       └─ 🖍 형광펜 (5색 선택 → 저장)
    │
    ├─▶ 단어 더블클릭 → 📖 정의 + 문맥 설명
    │
    ├─▶ [F] 집중 모드 진입
    │       ├─ [A] AI 채팅 오버레이
    │       └─ [Esc] 집중 모드 종료
    │
    └─▶ 메모(📝) / 북마크(★) 추가
```

### 여정 B — AI 어휘·퀴즈 생성 (범위 선택 포함)

```
[뷰어 헤더] 📚 어휘 생성  또는  🎯 퀴즈 생성  버튼 클릭
    │
    ▼
[RangeSelector 모달] 범위 선택
    ├─ 📚 책 전체
    ├─ 📖 현재 챕터  ← PDF 목차(getOutline) 기반
    │                  없으면 현재 페이지 ±10p 폴백
    └─ 🎯 커스텀 범위 (시작 페이지 ~ 끝 페이지)
    │
    ▼
[텍스트 확보 로직]
    ├─ 캐시 있음 → 즉시 사용
    └─ 캐시 없음 → PdfViewer.ensureRange() 호출
            ├─ getTextContent()로 텍스트 추출
            └─ 텍스트 없음(이미지 PDF) → AI Vision OCR
                    ├─ Cloud Vision API (apiKeys.vision 있을 때)
                    └─ Claude / Gemini Vision (폴백)
                    최대 20페이지 / 진행 토스트 표시
    │
    ▼
AI 호출 (Claude 또는 Gemini)
    ├─ 📚 어휘 → 중요 단어 5개 추출 → [지식 > 어휘] 저장
    │             성공 토스트: "✓ 어휘 N개 추가됨"
    └─ 🎯 퀴즈 → 5지선다 문제 생성 → QuizModal 즉시 표시
                  정답 선택 → 채점 + 해설
```

### 여정 C — 지식 체계화

```
[지식 🧠]
    ├─ 📝 노트 탭:  메모 + 하이라이트 목록 (태그로 필터)
    ├─ 🃏 카드 탭:  AI 플래시카드
    │                카드 클릭 → Q↔A 플립
    │                ✓(알겠음) / 🔄(다시 볼게) 마킹
    └─ 📚 어휘 탭:  뷰어에서 생성한 어휘 목록
```

### 여정 D — 독서 습관 & 목표

```
[목표 🎯]
    ├─ 🔥 독서 스트릭:  연속 일수 배지 + 월간 달력 (GitHub 잔디 스타일)
    ├─ ⏱ 독서 속도:    분당 페이지, 예상 완독일 계산
    ├─ 🔔 일일 알림:    브라우저 알림 (원하는 시각 설정)
    ├─ 🏆 완독 챌린지:  월간 N권 목표 + 조건 달성 시 배지 자동 지급
    └─ 🤖 AI 코치:      최근 7일 패턴 분석 → 맞춤 개선 제안
```

### 여정 E — AI 심화

```
[AI 🤖]
    ├─ 💬 채팅 탭
    │       ├─ 모드: 빠른 답변 / 맥락 분석 / 소크라테스
    │       ├─ 컨텍스트: 책 본문 + 메모 + 하이라이트 + 이미지(스캔 PDF)
    │       └─ Claude / Gemini 모델 선택 (양쪽 키 있을 때)
    │
    └─ 📊 책 비교 탭
            ├─ 책 A ↔ 책 B 선택 (서재 목록에서)
            ├─ 비교 관점: 종합 / 공통 주제 / 상반된 관점 / 독서 순서
            └─ 컨텍스트: 메타데이터 + 메모 + 하이라이트 + 본문 요약
```

### 여정 F — 내보내기 & 공유

```
[서재 📚] 책 상세 화면
    ├─ 📤 공유
    │       ├─ QR 코드 (앱 딥링크 기반)
    │       ├─ URL 복사: https://pkl-webapp.vercel.app/?book={fileId}
    │       ├─ Drive 링크 (공유 설정된 경우)
    │       └─ 네이티브 공유 (모바일 OS 공유 시트)
    ├─ 📄 노트 PDF 출력 (하이라이트 + 메모 → HTML → print)
    ├─ 📝 Markdown 다운로드
    ├─ 🎨 리뷰 카드 (1080×1080 PNG, 5종 테마, 별점)
    └─ 📁 컬렉션/책장 분류

[목표 🎯] ─▶ 📊 독서 통계 이미지 (월간/연간 인포그래픽)
[설정 ⚙]  ─▶ ☁️ Drive 자동 백업 (세션 종료 시 메모/하이라이트 → Drive)
```

## 5. 데이터 흐름 (BYOK 아키텍처)

```
┌──────────────┐      PDF 다운로드        ┌─────────────────┐
│   사용자      │ ◀──────────────────────  │  Google Drive   │
│  브라우저     │                          └─────────────────┘
│              │      텍스트 / 이미지       ┌─────────────────┐
│ (로컬 전용)   │ ──────────────────────▶  │  Claude/Gemini  │ ← 사용자 키
│              │ ◀──── AI 응답 ──────────  │   Vision OCR    │
└──────┬───────┘                          └─────────────────┘
       │
       ▼  localStorage (서버에 데이터 저장 없음)
  ┌────────────────────────────────────────────┐
  │  pkl_config          설정 · API 키 · Drive 토큰  │
  │  pkl_book_{id}       책별 진행률 · 메타데이터     │
  │  pkl_annot_{id}      형광펜 주석               │
  │  pkl_notes · cards · vocab · sessions · badges  │
  └────────────────────────────────────────────┘
```

> **BYOK (Bring Your Own Key)**: 모든 AI 호출은 사용자 본인의 API 키로  
> 브라우저에서 직접 수행됩니다. 서버에 키나 책 내용을 저장하지 않습니다.

## 6. 오프라인 지원 (PWA)

```
온라인  ─▶ 앱 셸 (JS · CSS · 폰트) 캐시에 저장
오프라인 ─▶ 앱 UI 실행 가능 ✅
           로컬 메모 · 노트 열람 가능 ✅
           PDF 로딩 ✗ (Google Drive 접근 불가)
           AI 기능 ✗ (API 호출 불가)
```

- Chrome/Edge PC: 주소창 설치 아이콘 클릭
- Safari/Chrome 모바일: 공유 → "홈 화면에 추가"
