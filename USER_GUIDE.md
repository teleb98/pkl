## 1. 시작하기

### Google Drive 연결
1. https://pkl-webapp.vercel.app 에 접속합니다.
2. **시작하기** → **Google 계정 연결**을 눌러 Drive 접근을 허용합니다.
3. PDF 책이 들어 있는 **폴더를 선택**합니다.

### AI 키 입력 (선택, 강력 권장)

설정(⚙) → API 키에서 입력합니다.

| 키 | 용도 |
|---|---|
| **Claude API 키** | AI 채팅, 어휘/퀴즈 생성, 책 비교, OCR |
| **Gemini API 키** | 동일 (Claude 없을 때 자동 사용) |
| **Cloud Vision API 키** | 스캔 이미지 PDF OCR (선택, 더 정확함) |

> 🔒 **개인정보 보호**: 모든 키와 책 내용은 **브라우저 localStorage**에만 저장됩니다.  
> 서버에는 아무것도 전송되지 않습니다 (BYOK 방식).

## 2. 책 읽기 (뷰어)

서재(📚)에서 책을 클릭하면 뷰어가 열립니다.

### 페이지 이동

| 방법 | 동작 |
|---|---|
| `←` `→` 키 또는 `PageUp/Down` | 이전/다음 페이지 |
| 마우스 휠 (페이지 끝에서) | 페이지 전환 |
| 좌우 스와이프 (모바일) | 이전/다음 페이지 |

### 읽기 설정 (⚙)
뷰어 헤더의 ⚙ 버튼 또는 집중 모드 툴바에서 접근합니다.

- **배경**: 기본(흰색) / 세피아 / 다크
- **확대**: 75% / 100% / 125% / 150%

### 집중 모드 `[F]`
- 헤더/사이드바를 숨기고 PDF에 집중
- 하단 플로팅 툴바: 페이지 이동, 북마크, AI, 설정
- `[A]` → AI 채팅 오버레이 열기
- `[Esc]` → 집중 모드 종료

## 3. 형광펜 · 메모 · 단어 검색

### 🖍 형광펜 주석
1. PDF 텍스트를 **드래그**해서 선택합니다.
2. 나타나는 팝업에서 **색상**(노란/초록/파란/분홍/보라)을 고릅니다.
3. **형광펜** 버튼을 누르면 저장됩니다.
4. 형광펜은 페이지를 재방문해도 유지됩니다.

### 💡 즉석 AI 질문
- 텍스트를 드래그하면 팝업에 AI 질문 버튼도 나타납니다.
- 선택한 내용에 대해 바로 AI에게 물어볼 수 있습니다.

### 📖 단어 정의 검색
- 단어를 **더블클릭**합니다.
- 해당 단어의 정의와 문맥 내 의미가 팝업으로 표시됩니다.

### 📝 메모 & ★ 북마크
- 메모 버튼(📝): 텍스트 메모 또는 하이라이트 추가, 태그 지원
- 북마크(★): 현재 페이지 즐겨찾기

## 4. AI 어휘 · 퀴즈 생성

뷰어 상단 헤더에서 바로 생성할 수 있습니다.

| 버튼 | 기능 |
|---|---|
| 📚 | **어휘 생성** — 해당 범위에서 중요 단어 5개 추출 후 [지식 > 어휘]에 저장 |
| 🎯 | **퀴즈 생성** — 5지선다 문제 생성 후 즉시 풀이 화면 표시 |
| 📖 | **지식 보기** — 지식 화면으로 바로 이동 |

### 범위 선택 옵션

버튼을 누르면 범위 선택 모달이 열립니다:

- **📚 책 전체** — 책에서 캐시된 전체 텍스트 사용
- **📖 현재 챕터** — PDF 북마크/목차를 자동으로 읽어 현재 페이지가 속한 챕터 전체  
  *(목차 없는 PDF는 현재 페이지 ±10쪽으로 자동 대체)*
- **🎯 커스텀 범위** — 시작 페이지~끝 페이지를 직접 입력

### 스캔(이미지) PDF 자동 OCR

텍스트가 없는 이미지형 PDF(스캔본)에서 어휘/퀴즈를 생성하면:
- AI Vision이 자동으로 글자를 인식합니다 (최대 20페이지)
- 진행 상황이 토스트 메시지로 표시됩니다
- Cloud Vision API 키가 있으면 더 정확한 OCR이 적용됩니다

> 💡 **팁**: 생성 전에 해당 범위를 한 번 스크롤해 넘겨보면  
> 텍스트가 미리 캐시되어 더 빠르게 처리됩니다.

## 5. 지식 체계화

지식(🧠) 화면 — 3개 탭:

### 📝 노트 탭
- 뷰어에서 추가한 **메모**와 **하이라이트** 목록
- 태그로 필터링 가능

### 🃏 카드 탭  
- **AI 플래시카드** — AI가 생성한 Q&A 카드
- 카드 클릭 → 앞면(질문) ↔ 뒷면(답변) 플립
- **✓** (알겠음) / **🔄** (다시 볼게) 학습 상태 마킹

### 📚 어휘 탭
- 뷰어에서 📚 버튼으로 생성한 어휘 목록
- 단어 + AI 생성 정의 표시

## 6. AI 채팅 & 책 비교 분석

AI(🤖) 화면 — 2개 탭:

### 💬 AI 채팅 탭

**채팅 모드 3종:**

| 모드 | 동작 방식 |
|---|---|
| ⚡ 빠른 답변 | 간결하고 직접적인 답변 |
| 🔗 맥락 분석 | 책 본문·메모·하이라이트를 깊이 분석 |
| 🏛 소크라테스 | 답을 주지 않고 생각을 유도하는 질문 방식 |

- 현재 책의 **본문 텍스트, 메모, 하이라이트**를 컨텍스트로 자동 활용
- 스캔 PDF는 현재 페이지 이미지를 보고 답변 (이미지 인식 모드)

### 📊 책 비교 탭

두 책을 AI로 심층 비교합니다:

1. **책 A**와 **책 B**를 서재 목록에서 선택합니다.
2. 비교 관점을 고릅니다:
   - **종합 비교**: 공통 주제, 차이점, 시너지, 독서 순서 전체
   - **공통 주제**: 두 책이 공유하는 핵심 개념과 주제
   - **상반된 관점**: 서로 다르거나 긴장 관계에 있는 주장
   - **독서 순서**: 어떤 책을 먼저 읽는 것이 좋은지 이유와 함께 제시
3. **분석 시작** 버튼을 누릅니다.

> 각 책의 메타데이터(저자·요약·주제), 메모, 하이라이트, 본문을 컨텍스트로 활용합니다.

## 7. 독서 습관 & 목표 관리

목표(🎯) 화면:

### 🔥 독서 스트릭
- 연속 독서 일수 (스트릭) 표시
- 월간 달력으로 독서한 날 시각화

### ⏱ 독서 속도 트래커
- 평균 독서 속도 (분/페이지)
- 현재 진행 중인 책의 예상 완독일

### 🔔 일일 독서 알림
- 원하는 시각에 브라우저 알림 설정
- 처음 설정 시 브라우저 알림 권한 허용 필요

### 🏆 완독 챌린지 & 배지
- 월간 목표 권수 설정
- 달성 조건 충족 시 배지 자동 지급

### 🤖 AI 독서 코치
- 최근 7일 독서 패턴(총 시간, 세션 수, 활동 일수) 분석
- AI가 맞춤 개선 제안 제공

## 8. 내보내기 & 공유

### 📤 책 공유 (서재 → 책 상세 → 공유 버튼)

- **QR 코드**: 모바일로 스캔하면 앱에서 해당 책으로 바로 이동
- **앱 딥링크**: `https://pkl-webapp.vercel.app/?book={ID}` URL 복사
- **Drive 링크**: 원본 PDF 파일 링크 (Drive 공유 설정된 경우)
- **네이티브 공유**: 모바일에서 OS 기본 공유 시트 사용

### 기타 내보내기

| 위치 | 기능 |
|---|---|
| 서재 → 책 상세 | **노트 PDF 출력** — 하이라이트+메모를 표지 포함 PDF로 인쇄 |
| 서재 → 책 상세 | **Markdown 다운로드** — 노트를 .md 파일로 저장 |
| 서재 → 책 상세 | **리뷰 카드** — 1080×1080 PNG 이미지 (5종 테마, 별점, 독서 통계) |
| 서재 → 책 상세 | **컬렉션/책장** — 책을 여러 컬렉션으로 분류·관리 |
| 목표 화면 | **독서 통계 이미지** — 월간/연간 인포그래픽 PNG |
| 설정(⚙) | **Drive 자동 백업** — 메모·하이라이트를 Drive에 자동 저장 |

## 9. 앱 설치 (PWA)

PKL은 PWA(Progressive Web App)로 설치할 수 있습니다.

- **PC (Chrome/Edge)**: 주소창 우측 설치 아이콘 클릭
- **모바일 (iOS Safari)**: 공유(□↑) → "홈 화면에 추가"
- **모바일 (Android Chrome)**: 더보기(⋮) → "앱 설치"

**설치 후 장점:**
- 홈 화면 아이콘으로 네이티브 앱처럼 실행
- 오프라인에서 앱 UI와 저장된 노트 열람 가능  
  *(PDF 로딩·AI 기능은 인터넷 연결 필요)*

## 10. 자주 묻는 질문

**Q. "AI 키를 먼저 설정하세요"라는 메시지가 나와요.**  
설정(⚙) → API 키에서 Claude 또는 Gemini 키를 입력하세요.

**Q. "페이지 텍스트가 아직 없습니다" 오류가 발생해요.**  
해당 범위 페이지를 한 번 넘겨본 뒤 다시 시도하세요.  
스캔 이미지 PDF는 AI Vision OCR이 자동으로 처리하지만, AI 키가 필요합니다.

**Q. "현재 챕터"가 이상한 범위로 잡혀요.**  
PDF에 목차(북마크) 정보가 없으면 현재 페이지 ±10쪽으로 자동 대체됩니다.  
정확한 챕터 인식은 목차가 포함된 PDF에서만 동작합니다.

**Q. PDF가 "network error"로 열리지 않아요.**  
설정에서 Google Drive를 다시 연결해 토큰을 갱신하세요.  
Drive 토큰은 1시간마다 자동 갱신되지만, 실패 시 수동 재연결이 필요합니다.

**Q. 내 데이터는 어디에 저장되나요?**  
모든 데이터는 브라우저 localStorage에만 저장됩니다.  
서버에는 아무것도 저장하지 않습니다. 중요한 노트는 **Drive 백업** 기능을 활용하세요.

**Q. 여러 기기에서 같이 쓰고 싶어요.**  
- 설정 → Drive 백업으로 메모/하이라이트를 동기화할 수 있습니다.
- 책 자체는 Drive에 있으므로 같은 폴더를 연결하면 어느 기기에서든 접근 가능합니다.
- 현재 읽기 진행률(페이지 위치)은 기기 간 동기화되지 않습니다.

**Q. 스캔(이미지) PDF도 어휘·퀴즈 생성이 되나요?**  
네, 됩니다. 텍스트가 없으면 AI Vision OCR로 자동 전환합니다.  
- AI 키(Claude/Gemini) 필수
- Cloud Vision API 키 추가 시 더 정확한 OCR 적용
- 범위당 최대 20페이지 처리 (시간이 다소 걸릴 수 있음)
