수집하는 정보

1. 수집하는 정보

PKL은 Google 로그인을 선택한 경우 다음 정보만 수집합니다:

  • 이름 및 이메일 주소 — 앱 내 프로필 표시 목적으로만 사용
  • 프로필 사진 — 앱 내 표시 목적으로만 사용

Google 로그인을 건너뛰면 어떤 정보도 수집하지 않습니다.

2. 수집하지 않는 정보

  • AI API 키 (Claude, Gemini) — 브라우저 로컬 저장소에만 저장, 서버 전송 없음
  • 메모, 하이라이트, 독서 기록 — 브라우저 로컬 저장소에만 저장
  • 독서 목표, 검색 기록 — 브라우저 로컬 저장소에만 저장
  • 결제 정보, 주민등록번호, 그 외 민감 정보

3. AI API 키 처리 방식

사용자가 입력한 Claude(Anthropic) 및 Gemini(Google) API 키는:

  • 오직 사용자 본인의 기기 브라우저(localStorage)에만 저장됩니다.
  • AI 요청 시 각 AI 서비스(Anthropic / Google)에 직접 전송됩니다. PKL 서버를 경유하지 않습니다.
  • API 사용 비용은 사용자 본인의 계정에서 발생합니다.

4. Google 서비스 연동

Google 로그인을 선택한 경우:

  • 요청 권한: 기본 프로필 정보(이름, 이메일, 프로필 사진)만 요청합니다.
  • Google Drive에 대한 접근 권한을 요청하지 않습니다.
  • 수집된 Google 프로필 정보는 서버에 저장되지 않으며, 브라우저 세션 동안만 사용됩니다.

5. 제3자 제공

수집된 정보를 제3자에게 판매하거나 공유하지 않습니다.

단, AI 기능 사용 시 입력한 내용은 사용자가 선택한 AI 서비스(Anthropic 또는 Google)로 직접 전송됩니다. 해당 서비스의 개인정보처리방침을 함께 확인하세요.

6. 데이터 삭제

앱 설정(⚙) → "앱 초기화"를 통해 브라우저에 저장된 모든 데이터를 즉시 삭제할 수 있습니다.

📌 Summary: This app has no server. All data (notes, highlights, API keys) is stored only in your browser and never sent to our servers.

What we collect

1. Information We Collect

PKL only collects the following if you choose to sign in with Google:

  • Name and email address — used only to display your profile in the app
  • Profile picture — used only for display purposes

If you skip Google sign-in, no information is collected.

2. Information We Do NOT Collect

  • AI API keys (Claude, Gemini) — stored only in your browser's local storage, never transmitted to our servers
  • Notes, highlights, and reading history — stored only in your browser's local storage
  • Reading goals, search history — stored only in your browser's local storage
  • Payment information or any other sensitive data

3. How API Keys Are Handled

Claude (Anthropic) and Gemini (Google) API keys you enter are:

  • Stored only in your device's browser (localStorage).
  • Sent directly to the respective AI service (Anthropic / Google) when you use AI features. They do not pass through PKL servers.
  • Any API usage costs are charged to your own account.

4. Google Services

If you choose to sign in with Google:

  • We only request basic profile information (name, email, profile photo).
  • We do not request access to Google Drive.
  • Google profile data is not stored on any server and is only used during your browser session.

5. Third-Party Sharing

We do not sell or share any collected information with third parties.

When you use AI features, your messages are sent directly to the AI service you've selected (Anthropic or Google). Please review their privacy policies as well.

6. Deleting Your Data

You can delete all locally stored data at any time via Settings (⚙) → "Reset App".
