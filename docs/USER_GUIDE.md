# PKL 사용 설명서 (Personal Knowledge Library)

> **PKL**은 내 책(PDF)을 AI와 함께 읽고, 메모·하이라이트로 지식을 쌓는 독서 앱입니다.
> 웹·데스크톱·태블릿 어디서나 같은 경험으로 쓸 수 있고, **책 내용이 기기를 벗어나지 않는 로컬 우선** 설계입니다.
> 버전 1.0.0 · 최종 업데이트 2026-06-07

## 1. 한눈에 보기

| 기능 | 설명 |
|------|------|
| 📚 **내 기기 PDF** | 기기에 저장된 PDF를 추가해 오프라인으로 읽기 |
| ☁️ **Google Drive 책** | Drive 폴더의 PDF를 불러와 읽기 (선택) |
| 📖 **뷰어** | 페이지 넘김·확대·형광펜·북마크·집중 모드 |
| 🤖 **AI 분석** | 책 내용을 읽고 요약·질의응답 (빠른/맥락/소크라테스) |
| 🔤 **로컬 OCR** | 스캔본 PDF 글자를 기기 내에서 인식 (Tesseract/Gemma 4) |
| 📝 **지식** | 메모·하이라이트·AI 어휘·퀴즈를 한곳에 |
| 🎯 **목표** | 독서 목표·습관 추적 |
| 📤 **내보내기·공유** | 노트를 PDF/Markdown/HTML, QR 공유 |

**4개 플랫폼, 하나의 코드**: 웹(브라우저) · 데스크톱(Mac/Win/Linux) · iPad · Android 태블릿

## 2. 설치 — 플랫폼별

### 🌐 웹 (가장 빠름)
- 브라우저에서 **https://pkl-webapp.vercel.app** 접속, 설치 불필요
- iPad/모바일은 Safari **공유 → "홈 화면에 추가"**로 앱처럼 사용 (PWA)

### 🖥 데스크톱 (Mac / Windows / Linux)
- **macOS**: `PKL …-arm64.dmg`(Apple Silicon) / `…-x64.dmg`(Intel) → Applications로 드래그
  - 첫 실행 시 경고 → **우클릭 → 열기** (1회만)
- 네이티브 파일 선택·메뉴·완전 오프라인 지원

### 📱 Android 태블릿 (갤럭시 탭 등)
- `PKL-galaxytab-release-v1.0.0.apk`를 탭으로 전송(카톡/이메일/드라이브)
- 탭에서 APK 터치 → "이 출처 허용" → 설치 (Android 7.0+)

### 🍎 iPad
- 현재는 **PWA(홈 화면 추가)** 권장. 네이티브 빌드는 Xcode 필요

## 3. 처음 시작 (온보딩)

```
[1] Google      [2] AI 키       [3] 완료
 계정 연결        선택 입력         시작
```
- **웹**: Google 계정 연결 시 프로필 + Drive 연동 준비
- **데스크톱/태블릿**: **"앱으로 바로 시작"** → **"로컬 PDF로 시작하기"** (로그인 불필요)
- **AI 키**: "나중에 설정" 가능

> 💡 로그인 없이도 내 기기 PDF 읽기·메모·하이라이트가 모두 동작합니다.

## 4. 책 추가하기

### 방법 A — 내 기기 PDF (권장, 오프라인)
1. 서재 상단 **"책 추가"** 또는 **"내 기기 > + PDF 추가"**
2. **"PDF 파일 가져오기"** → 파일 선택
3. 즉시 서재 등록 + 뷰어로 열림 (⚡캐시됨 표시 → 오프라인 즉시 로드)

### 방법 B — Google Drive (선택)
- 설정에서 Drive 폴더 연결 → 폴더의 PDF가 서재에 자동 표시
- 첫 열기 1회 다운로드 후 캐시

| 동작 | 방법 |
|------|------|
| 페이지 넘김 | 좌/우 스와이프·클릭·화살표 |

| 진도 기록 | 상단 `− p.123 +` |
| **집중 모드** | ✦ / `F` 키 — 몰입 읽기 + AI 사이드 |
| 배경색 | 읽기 설정(화이트/세피아 등) |

## 6. AI 분석 — 책 기반 질문

AI가 **실제 책 내용을 읽고** 답합니다 (제목만 보고 답하지 않음).

**3가지 모드**
- ⚡ **빠른 답변** · 🔗 **맥락 분석** · 🏛 **소크라테스**(질문으로 사고 유도)

**사용**: AI 대화 화면에서 질문, 또는 추천 질문(핵심 요약/저자 주장/실용 교훈/비판적 시각) 탭

> 작동 원리: 질문 시 책 텍스트가 자동 추출돼 AI에 함께 전달됩니다. 뷰어를 안 거쳐도, 앱을 재시작해도 책 기반 답변이 보장됩니다.

**AI 키**: 설정에서 Gemini(무료 티어 가능) 또는 Claude 키. 비용은 본인 키로 직접 과금(BYOK).

## 7. 스캔 PDF OCR

이미지로만 된 스캔본도 글자를 인식해 AI·검색·어휘에 활용합니다.
**설정 → "스캔 PDF OCR 방식"**:

| 모드 | 동작 | 특징 |
|------|------|------|
| **자동** | 로컬 우선 → 클라우드 폴백 | 기본·권장 |
| **로컬** | 기기 내에서만 | 🔒 프라이버시·오프라인·무료 |
| **클라우드** | Cloud Vision / AI | 최고 정확도 (키 필요) |

**로컬 엔진** (자동 선택):
- **Tesseract** — 모든 기기 내장, 한글+영문, 즉시
- **Gemma 4 (Ollama)** — 데스크톱, 표·수식·손글씨 강함 (`ollama pull gemma4:e4b`)
- **Gemma 4 (브라우저/WebGPU)** — 웹·태블릿, 모델 URL 설정 시(고급)

## 10. Google Drive 연동 (선택)
- 설정 → Google 계정 연결 → Drive 폴더 선택 (클라우드 책·백업)
- **데스크톱 앱**은 별도 OAuth 설정 필요 → `ELECTRON_OAUTH_SETUP.md`
- 미설정 시에도 내 기기 PDF로 전체 기능 사용 가능

## 11. 설정
- **테마**: Ember/Sage/Ink + 다크 모드 · **글꼴**: Lora/Newsreader/Cormorant

## 12. 프라이버시·보안
- **BYOK**: API 키는 본인 기기에만 저장, 서버 전송 없음
- **로컬 우선**: 내 기기 PDF·로컬 OCR은 클라우드 전송 없음
- **저장**: 메모·진도는 localStorage, PDF는 기기 캐시(IndexedDB)
- **보안 헤더(CSP 등)**: 외부 스크립트·임의 파일 접근 차단
- **앱 초기화**: 설정 하단에서 전체 데이터 삭제

## 13. FAQ

**로그인 꼭 해야 하나요?** — 아니요. 데스크톱·태블릿은 로그인 없이 시작.
**AI 키 없으면?** — 읽기·메모·로컬 OCR은 동작. AI 질문·어휘·퀴즈만 키 필요(Gemini 무료 가능).
**오프라인 가능?** — 내 기기 PDF + 로컬 OCR은 완전 오프라인. AI/Drive/클라우드 OCR만 네트워크.
**스캔 책도 AI가 읽나요?** — 네, OCR로 글자 인식 후 AI가 그 텍스트로 답.
**갤럭시 탭 "출처 허용" 경고?** — 정상, 1회 허용.
**내 책이 외부로 나가나요?** — 로컬 모드는 안 나감. 클라우드 OCR/AI 사용 시에만 해당 페이지 전송.
